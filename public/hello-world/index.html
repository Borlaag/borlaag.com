Hello, World.

October 1, 2022

The mandatory first post in which I introduce this new blog along with some details on what tools and other resources are used to make it. Also included are details about relevant content and code licenses.

My goals with this blog are fairly unstructured, but mostly I want a place to put my thoughts and designs on primarily software related stuff.

Stack

This blog is a static site generated using Zola, a static website generator written in Rust. I'm using a slightly modified version of the zola-paper theme.

Some of the modifications include subtle colors customizations the use of the Inter font for textand Fira Code for code.

The website is currently deployed to Netlify, though I plan to self-host once I get my server up and running. I'll likely write a separate post about that in the near future.
